# Story-On UI Specification

## 1. 개요
Story-On은 창작자들을 위한 AI 보조 웹사이트입니다. 본 UI 사양서는 웹 애플리케이션의 기본적인 페이지 구조, 컴포넌트 설계 및 라우팅 구조를 정의합니다.

## 2. 공통 UI 요소 (글로벌 레이아웃)
웹사이트의 모든 페이지에서 유지되는 공통 구조는 다음과 같습니다.

### 2.1 상단 네비게이션 바 (GNB)
- **위치**: 화면 상단 고정 (Fixed Sticky)
- **왼쪽**: 프로젝트 로고 (Story-On)
- **가운데 (메인 메뉴)**:
  - Home
  - Main
  - CC AI
  - SCS AI
  - ETC
  - *동작*: 각 버튼 클릭 시 해당 페이지로 이동
- **오른쪽**: "내 작업실" 버튼 (아이콘 + 텍스트 형태)

## 3. 페이지별 상세 사양

### 3.1 Home 화면 (`/`)
사용자가 사이트에 처음 진입했을 때 만나는 랜딩 페이지입니다.
- **Hero 섹션**: Story-On 브랜드를 강조하는 큰 타이틀과 소개 문구
- **시각 요소**: 창작 영감을 주는 동적인 애니메이션/일러스트 (디자인 시스템 적용)
- **도구 소개 카드**: CC AI, SCS AI 등 주요 기능에 대한 간략한 설명 카드 그리드
- **네비게이션 요소**: 맨 아래로 스크롤 시 Main 페이지로 진입할 수 있는 "Next" (또는 시작하기) 버튼

### 3.2 Main 화면 (`/main`)
사용자가 본격적으로 도구를 선택하는 환승 허브 역할의 뷰포트입니다.
- **구조**: 화면 중앙에 3개의 큰 카드/버튼 (CC AI, SCS AI, ETC) 배치
- **시각 요소**: 각 버튼 근처나 내부에 해당 기능을 상징하는 간단한 애니메이션/시각적 요소 포함

### 3.3 CC AI 화면 (`/cc-ai`)
Character Create AI 도구를 활용하는 작업 공간입니다.
- **기존 캐릭터 입력 영역**: 사용자가 구상 중인 캐릭터의 배경설정이나 특징을 텍스트로 입력
- **메타데이터 표시 영역**: 캐릭터 속성을 여러 축(예: 성격, 능력치, 가치관 등)으로 분해하여 시각적으로 표시(다이어그램 또는 태그 형태)
- **키워드 선택 영역**: AI가 추천하는 키워드 또는 사용자가 원하는 방향성의 키워드 풀
- **결과 영역**: AI를 통해 구체화/새롭게 생성된 캐릭터 결과물 표시

### 3.4 SCS AI 화면 (`/scs-ai`)
Scenario Choice Simulation AI 도구를 활용하는 작업 공간입니다.
- **스토리/상황 입력 영역**: 현재까지 진행된 이야기 또는 놓여있는 상황을 서술
- **시뮬레이션 옵션 생성 영역**: AI에게 다양한 선택지 생성을 요청하는 컨트롤 패널
- **결과 비교 영역**: 각 시뮬레이션 결과(선택에 따른 잠재적 미래/결과)를 가로 스크롤이나 카드 형태로 나란히 배치하여 비교

### 3.5 ETC 화면 (`/etc`)
기타 보조/확장 도구들을 제공하는 메뉴입니다.
- **도구 목록**: 사용 가능한 도구들을 카드 형태의 그리드 레이아웃으로 표시 (ex. 세계관 생성기, 이름 생성기 등)

### 3.6 내 작업실 모달/플라이아웃 창 (`/studio` 또는 Drawer)
글로벌 네비게이션에서 접근할 수 있는 개인 보관함입니다. (오른쪽 사이드바(Drawer) 형태로 구현 권장)
- **저장한 캐릭터**: CC AI에서 저장한 결과물 목록
- **저장한 시나리오**: SCS AI에서 저장한 결과물 목록
- **최근 작업**: 모든 도구 사용 기록 중 가장 최근 항목들 표시

## 4. 컴포넌트 설계

### 4.1 Layout Components
- `Header`: 글로벌 네비게이션 바
- `Footer`: 하단 정보 (Home 페이지 전용으로 사용 또는 전체 공통으로 사용 가능)
- `PageContainer`: 상단 고정 헤더를 고려한 컨텐츠 래퍼

### 4.2 Shared UI Components
- `Button`: 기본/강조/텍스트 등 다양한 변형을 지원하는 버튼 컴포넌트
- `Card`: 도구 소개, 결과물 표시에 사용되는 컨테이너
- `Input` / `TextArea`: 정보 입력 폼 요소
- `Tag` / `Badge`: 키워드나 상태 표시
- `AnimatedIcon`: 기능을 상징하는 애니메이션 요소 아이콘

### 4.3 Feature Components
**CC AI 전용**
- `CharacterInputForm`
- `MetadataRadarChart` (또는 축 표시기)
- `KeywordSelector`
- `CharacterResultCard`

**SCS AI 전용**
- `ScenarioContextInput`
- `SimulationOptionPanel`
- `ScenarioComparisonList`

**Studio 전용**
- `StudioDrawer`: 내 작업실 열림/닫힘 UI
- `SavedItemCard`: 저장된 에셋 미리보기 카드

## 5. 라우팅 구조

React Router를 사용한 라우팅 체계 (SPA 구조):

- `/` : 홈 화면 (`Home` 컴포넌트)
- `/main` : 메인 기능 선택 화면 (`Main` 컴포넌트)
- `/cc-ai` : Character Create AI 도구 화면 (`CCAI` 컴포넌트)
- `/scs-ai` : Scenario Choice Simulation AI 도구 화면 (`SCSAI` 컴포넌트)
- `/etc` : 기타 도구 화면 (`ETC` 컴포넌트)

*참고: "내 작업실"은 별도 페이지 이동 없이 상태 기반의 Drawer/Modal(오른쪽에서 나타나는 패널)로 구현하여 맥락 훼손 없이 접근 가능하도록 함.*
